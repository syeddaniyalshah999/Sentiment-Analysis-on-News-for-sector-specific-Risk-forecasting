# Stock Market Volatility Forecasting Using Sentiment Analysis and Machine Learning

Can market sentiment and macroeconomic factors help forecast risk in financial markets?

This project explores stock market volatility forecasting using News Sentiment, Twitter Sentiment, Gold Prices, and Exchange Rates through Machine Learning models including Linear Regression, Random Forest, and Support Vector Regression (SVR).

Unlike forecasting individual stock prices, sector-level volatility forecasting focuses on understanding and predicting overall market risk, making it more relevant for portfolio management, risk assessment, asset allocation, and investment decision-making.

The study compares sentiment-based and macroeconomic-based datasets and demonstrates how longer historical data and economic indicators can significantly improve volatility prediction performance.
